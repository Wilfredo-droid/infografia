Emisor

La fuente de la información el productor del mensaje comunicado

<--->

Contexto

El contexto es todo aquello a lo cual se refiere el mensaje. y de esta manera el contexto es toda la informacion alrededor de el mensaje necesario para comprender el mismo

Mensaje

El mensaje sirve como mediador ente el emisor y el receptor

Codigo

El código es un sistema lingüístico que se compone de elementos y reglas organizados. Este sistema es necesario para organizar y comprender los mensajes lingüísticos

Canal

El canal de comunicación existe en toda situacion de comunicacion, sirve para el intercambio de información entre un emisor y uno o más receptores: por ejemplo, el canal puede ser una llamada telefónica, correo electrónico, carta, señal de humo, red social, entre otros 

<--->

Receptor

Aquel que interpreta el mensaje
